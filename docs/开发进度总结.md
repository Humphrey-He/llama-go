## 核心架构

```
Client
  ↓
API Layer (internal/api)
  ├─ /v1/chat/completions (OpenAI 兼容)
  ├─ /v1/models (模型列表)
  ├─ /generate (原有 API)
  └─ /healthz, /readyz
  ↓
Model Registry (internal/model)
  └─ 模型注册与查询
  ↓
Backend Adapter (internal/backend)
  ├─ VLLMBackend (vLLM 后端)
  ├─ PythonBackend (Python 推理后端)
  └─ MockBackend (测试后端)
  ↓
Stream Module (internal/stream)
  └─ SSE 流式传输
  ↓
Session Store (internal/session)
  └─ 多轮对话存储（TTL + LRU）
```
