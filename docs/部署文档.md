# 部署文档

## 环境要求

- Docker & Docker Compose
- Go 1.25+ (本地开发)
- Python 3.10+ (本地开发)

## Docker 部署（推荐）

### 1. 一键启动
```bash
docker-compose up --build
```

### 2. 服务访问
- Go API 服务: `http://localhost:8080`
- Python 推理服务: `http://localhost:8000`

### 3. 停止服务
```bash
docker-compose down
```

## 本地开发部署

### 启动 Python 推理服务
```bash
cd python
pip install -r requirements.txt
python kv_server.py
```

### 启动 Go API 服务
```bash
go mod download
go run main.go
```

## 测试部署

```bash
# 测试接口
curl -X POST http://localhost:8080/api/generate \
  -H "Content-Type: application/json" \
  -d '{"prompt": "Hello"}'
```

## 配置说明

### 端口配置
修改 [internal/config/config.go](../internal/config/config.go):
```go
const (
    PythonBackendURL = "http://localhost:8000/infer"
    GoServerPort     = ":8080"
    MaxCacheSize     = 1000
)
```

### 模型配置
修改 [python/kv_server.py](../python/kv_server.py):
```python
MODEL_NAME = "TinyLlama/TinyLlama-1.1B-Chat-v1.0"
```

## 故障排查

### Python 服务无法启动
- 检查依赖: `pip list | grep transformers`
- 检查端口占用: `lsof -i :8000`

### Go 服务连接失败
- 确认 Python 服务已启动
- 检查配置中的 URL 是否正确
