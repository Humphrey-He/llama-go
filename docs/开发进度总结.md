## 核心架构

```
Client
  ↓
API Layer (internal/api)
  ├─ /v1/chat/completions (流式/非流式)
  ├─ /v1/sessions/{id}/clear
  └─ /healthz, /readyz
  ↓
Service Layer (internal/service)
  ├─ 上下文管理
  ├─ 消息历史
  └─ 后端调度
  ↓
Backend Adapter (internal/backend)
  ├─ vLLM
  └─ llama.cpp (待实现)
  ↓
Session Store (internal/session)
  └─ 多轮对话存储
```
