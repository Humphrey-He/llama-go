# Llama-Go 项目说明

## 项目概述

Llama-Go 是一个基于 Go + Python 的轻量级 LLM 推理服务，实现了 KV Cache 缓存优化，用于加速大语言模型的推理性能。

## 核心特性

- **双语言架构**：Go 负责高并发 API 服务，Python 负责模型推理
- **KV Cache 优化**：缓存 Transformer 模型的 Key-Value 状态，避免重复计算
- **并发安全**：使用 Go 的读写锁机制保证缓存的线程安全
- **轻量模型**：默认使用 TinyLlama-1.1B 模型，快速启动和推理

## 项目目标

1. 演示 Go 在 AI 服务中的高并发优势
2. 实现 KV Cache 机制提升推理效率
3. 提供简洁的 RESTful API 接口
4. 支持 Docker 容器化部署

## 应用场景

- 对话系统的上下文缓存
- 文本生成服务
- AI 推理性能优化研究
- Go + Python 混合架构实践

## 快速开始

```bash
# 启动服务
docker-compose up

# 测试接口
curl -X POST http://localhost:8080/api/generate \
  -H "Content-Type: application/json" \
  -d '{"prompt": "Hello, how are you?"}'
```
