# 本地部署指南

## 环境要求

- Windows 10/11
- Python 3.10+
- NVIDIA GPU（可选，推荐）
- 8GB+ 内存

## 快速开始

### 1. 安装依赖

在项目根目录，以管理员身份运行 PowerShell：

```powershell
.\setup_vllm.ps1
```

安装过程会：
- 创建 Python 虚拟环境
- 安装 PyTorch (CUDA 12.1)
- 安装 llama-cpp-python（支持 GGUF 格式，GPU 加速）
- 验证 GPU 可用性

### 2. 启动后端服务

```powershell
.\venv\Scripts\Activate.ps1
python python/start_llama_cpp.py
```

等待模型加载完成（约 30 秒），看到 `Uvicorn running` 表示启动成功。

### 3. 启动 Go 服务

新开一个终端：

```bash
go run cmd/inference/main.go
```

服务运行在 `http://localhost:8080`

## 验证部署

```bash
# 检查模型列表
curl http://localhost:8080/v1/models

# 测试对话
curl http://localhost:8080/v1/chat/completions \
  -H "Content-Type: application/json" \
  -d '{
    "model": "tinyllama-chat",
    "messages": [{"role": "user", "content": "Hello"}]
  }'
```

## 故障排查

### GPU 不可用

如果 PyTorch 检测不到 GPU：

```powershell
# 检查 CUDA 版本
nvidia-smi

# 重新安装 PyTorch (CUDA 11.8)
pip install torch --index-url https://download.pytorch.org/whl/cu118
```

### vLLM 安装失败

```powershell
# 使用 CPU 版本
pip install torch --index-url https://download.pytorch.org/whl/cpu
pip install vllm
```

### 端口占用

修改 `python/start_vllm_local.py` 中的端口号，同时更新 `cmd/inference/main.go` 中的 `BACKEND_URL`。

## Docker 部署（备选）

如果不需要 GPU 加速，可使用 Docker：

```bash
docker-compose up -d
```

注意：Docker 方式无法使用宿主机 GPU。
